📌 naszGPT — Customizable AI Chatbot (Streamlit)

⚡ naszGPT to interaktywna aplikacja AI (Streamlit), która pozwala:

rozmawiać z chatbotem opartym o LLM,

wybierać różne osobowości / tryby rozmowy,

zapisywać historię czatów,

oszacować koszty tokenów/wywołań,

testować AI w przyjaznym interfejsie.

To praktyczny projekt AI, pokazujący:
👉 integrację LLM z UI,
👉 workflow czatowy z kontekstem,
👉 logikę sesji i kosztów,
👉 produktowy charakter prostego narzędzia.

🧠 Funkcjonalności

✔ Chatbot oparty na modelach LLM
✔ Wybór osobowości (np. różne style odpowiedzi)
✔ Zachowywanie historii rozmów
✔ Liczenie/wizualizacja kosztów tokenów
✔ Działa jako lokalna aplikacja Streamlit

🛠️ Technologie

🐍 Python

⚡ Streamlit

🤖 OpenAI API / LLM

💾 JSON (do historii)
